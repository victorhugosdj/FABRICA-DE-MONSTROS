# Geral Informativo e Anexos

**O que é esta pasta?**
Esta é a biblioteca central de recursos administrativos e templates da equipe.

**O que você encontra aqui:**
- **Modelos de Documentos:** Templates padronizados (como o `MODELO DE DAILY.md`).
- **Comunicados:** Informações oficiais e avisos gerais.
- **Anexos:** Arquivos de suporte que não se encaixam em projetos específicos.

## 🛠️ Como Utilizar este Repositório

* **Novos Integrantes:** Comece lendo o guia de [Onboarding](https://www.google.com/search?q=ONBOARDING_NOVOS_COLABORADORES.md) para configurar sua pasta na seção `4 - EQUIPE`.
* **Rotina Diária:** Utilize o [Modelo de Daily](https://www.google.com/search?q=1%2520-%2520GERAL%2520INFORMATIVO%2520E%2520ANEXOS/MODELO%2520DE%2520DAILY.md) para documentar seu progresso.
* **Consulta:** O [Mapa de Conhecimento](https://www.google.com/search?q=MAPA_DE_CONHECIMENTO.md) serve como o índice central de todos os documentos.

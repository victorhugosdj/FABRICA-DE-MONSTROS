# Gestão

**O que é esta pasta?**
O centro de comando da equipe. Aqui organizamos nossos processos, rituais e planos.

**Estrutura:**
- `ATAS E REUNIOES`: Memória das decisões tomadas.
- `AVALIACAO DE RETRO`: O que aprendemos e como podemos melhorar.
- `PLANEJAMENTO SEMANAL`: O foco da semana.
- `ROADMAP`: Para onde estamos indo a longo prazo.

**Público-alvo:**
Todos os membros da equipe devem consultar esta pasta para alinhar prioridades.

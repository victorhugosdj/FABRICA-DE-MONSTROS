# Equipe

**O que é esta pasta?**
O espaço individual de cada colaborador.

**Como funciona:**
Cada membro tem sua própria subpasta (ex: `/DUDU`, `/KAUA`) contendo:
- `DAILIES`: Registro diário de atividades (Baseado no Modelo de Daily).
- `ANOTAÇOES PESSOAIS`: Espaço livre para rascunhos e estudos.

**Regra de Ouro:**
Respeite o espaço do colega, mas lembre-se que o conhecimento aqui também é público.

# Projetos

**O que é esta pasta?**
O coração da execução técnica. Aqui vive a documentação de tudo o que estamos construindo.

**Estrutura por Projeto:**
Cada projeto (ex: `PROJETO - FORMAÇAO MONSTRA`) possui:
- `DOCS`: Documentação técnica, arquitetura e manuais.
- `TAREFAS`: Backlog, kanban e gestão de atividades.

**Como contribuir:**
Mantenha a documentação técnica atualizada junto com o código.
